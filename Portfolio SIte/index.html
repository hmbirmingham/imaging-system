Automated Bacterial Colony Counter
End-to-end embedded imaging system built on Raspberry Pi for automated quantification of bacterial colonies. Combines controlled LED illumination hardware, a custom power management PCB, and a Python-based computer vision pipeline. Features two-layer anomaly detection using statistical methods and a Random Forest classifier, with automatic CSV logging and a touch display GUI. Targeting sub-5% quantification error benchmarked against manual laboratory workflows.

STM32 Bare-Metal Peripheral Drivers
Register-level peripheral drivers written from scratch on the STM32F411RE Nucleo board with no HAL abstraction. Each driver directly addresses hardware peripheral registers via memory-mapped I/O and is verified with a logic analyzer. Covers GPIO, UART, I2C, and SPI with clean documentation and build evidence for each.
C STM32 ARM Cortex-M4 ARM GCC OpenOCD st-flash Logic Analyzer

Two-Compartment Pharmacokinetic Simulator
Python-based simulator modeling drug concentration over time using a two-compartment PK model. Supports IV bolus, IV infusion, and oral dosing routes. Calculates AUC, Cmax, Tmax, and half-life. Validated against published pharmacokinetic parameters for real drugs.
